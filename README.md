\# AI Call Intelligence Dashboard



An AI-powered web application that:



\- Uploads customer call recordings

\- Converts speech to text using Whisper (offline)

\- Analyzes conversations

\- Extracts intent, budget, and urgency

\- Scores leads (Hot / Warm / Cold)



\## Tech Stack



\- Python

\- FastAPI

\- Whisper AI (offline speech-to-text)

\- TailwindCSS

\- Jinja Templates



\## Features



\- Audio Upload (.mp3 / .wav / .mp4)

\- Automatic Transcription

\- Lead Scoring System

\- Budget \& Timeline Extraction

\- Sales Insight Dashboard
